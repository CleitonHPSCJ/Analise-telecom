# Análise de Dados de Telecom - Operadoras Indianas

## Objetivo
Analisar o desempenho de operadoras de telecom na Índia, identificando padrões de rentabilidade, perfil de clientes e oportunidades de mercado.

## Ferramentas Utilizadas
- SQL (DBeaver)
- Banco de dados SQLite

## Principais Descobertas

### 1. Perfil de Idade e Gênero
- Todas as operadoras têm idade média de ~46 anos
- BSNL tem a maior proporção de clientes mulheres (~40%)

### 2. Rentabilidade por Operadora
- Reliance Jio é a operadora com maior rentabilidade
- O lucro está concentrado em Chhattisgarh (1.160 clientes)

### 3. Análise de Planos
- Airtel lidera em clientes Premium
- Reliance Jio domina com plano básico

## Conclusão Estratégica
A Reliance Jio domina em Chhattisgarh não por preço (que é igual ao das concorrentes), mas por fidelização e presença de mercado. Recomenda-se investimento em marketing de confiança e diferenciais qualitativos.

## Como Executar
1. Baixe o banco de dados
2. Execute os scripts SQL na ordem indicada
